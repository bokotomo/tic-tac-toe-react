props.

## squareSize

```
<TicTacToe squareSize={2} />
<TicTacToe squareSize={5} />
<TicTacToe squareSize={10} />
```

## text

```
<TicTacToe
    title="custom title"
    startButtonTitle="start button title"
    textWinMe="You Win!"
    textWinEnemy="You Lose!"
    textDraw="Draw!"
/>
```

## color

```
<TicTacToe itemColor="orange" itemMarkColor="red" />
```
